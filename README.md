# MERN Starter
Starter/seed project for MongoDB, Express, React + Mobxjs, Node full-stack JavaScript apps.

# installation
- docker
- install node, NPM
- run `npm install` in `server` and `client` folder

# how to start project
open terminal 1
```
docker-composer up
```

open terminal 2
```
cd server
npm run dev
```

open terminal 3
```
cd client
npm run start
```
